# Current Design

This document captures the game as it is currently implemented in the codebase. It is not a forward-looking pitch doc. It is a practical snapshot of the current playable design, grouped into:

1. Current design overview
2. Feature checklist
3. Progression and unlock map

## 1. Current Design Overview

### High-Level Concept

The game is a finance-themed incremental/idle management game where the player grows from a solo trader into a multi-layered trading organization with researchers, political influence, machine infrastructure, and prestige progression.

The implemented loop is:

`Manual trading -> early hires -> trader scaling -> research unlocks -> institutional desks -> bots and infrastructure -> lobbying -> prestige`

### Core Resources

- Cash: main economy currency used for units, upgrades, and infrastructure
- Research Points: used to unlock research tech and some repeatable optimizations
- Influence: generated by politicians and spent on lobbying policies and some repeatables
- Reputation: prestige currency used for permanent meta progression

### Core Gameplay Systems

#### Manual Trading

- Primary active action is `Trade`
- Generates cash instantly
- Scales through manual upgrades, policies, prestige, and repeatable optimizations
- Recent trade feedback is surfaced in the UI

#### Passive Income

Three passive production channels are implemented:

- Cash per second from human and machine trading units
- Research per second from scientist units
- Influence per second from politician units

#### Human Trading Ladder

Implemented trading units:

- Intern
- Junior Trader
- Senior Trader
- Prop Desk
- Institutional Desk
- Hedge Fund
- Investment Firm

These form the main human-side progression ladder from early game to late human-run scale.

#### Machine Trading Ladder

Implemented machine units:

- Rule-Based Bot
- ML Trading Bot
- AI Trading Bot

Machine trading is gated by research and infrastructure, and becomes a major late-run scaling layer.

#### Research Team

Implemented research staff:

- Intern Scientist
- Junior Scientist
- Senior Scientist

Scientists generate Research Points, which unlock major progression systems and feed research repeatables.

#### Politics / Lobbying

Implemented political staff:

- Senator

This unit generates Influence, which is used for lobbying policies and influence-funded repeatables.

#### Infrastructure / Power

Implemented infrastructure:

- Server Rack
- Server Room
- Data Centre
- Cloud Infrastructure

Power capacity and usage are tracked continuously.

- Units consume power
- Capacity comes from infrastructure
- If usage exceeds capacity, machine efficiency is reduced rather than fully shut off
- Infrastructure also gates several higher-tier unit unlocks

#### Upgrades

Two upgrade layers are implemented:

- One-time upgrades that unlock or boost specific systems
- Repeatable optimizations that scale output, reduce costs, expand capacity, or reduce power usage

#### Research Tree

Research is presented as a node-based dependency map.

- Unlocks major systems and unit tiers
- Supports visual dependency browsing
- Includes confirmation before purchase
- Has a larger modal version for expanded viewing

#### Prestige

Prestige is fully implemented as a reset-based meta progression loop.

- Requires sufficient lifetime cash and at least one Rule-Based Bot
- Grants Reputation
- Resets run-state progression
- Preserves prestige progression and permanent upgrades
- Supports planning prestige purchases before reset

### Main Screens / Tabs

Implemented main tabs:

- Desk
- Upgrades
- Optimizations
- Research
- Lobbying
- Prestige
- Stats
- Settings

Desk subviews:

- Trading
- Scientists
- Infrastructure
- Politicians
- Commodities (placeholder only, disabled)

### Modal Surfaces

Implemented modals:

- Save / Import
- Offline Earnings
- Prestige Confirm
- Reset Confirm
- Research Map

### Persistence / Utility

Implemented utility and save features:

- Autosave every 15 seconds
- Save on unload
- Manual save
- Export save string
- Import save string
- Offline progress with 8-hour cap
- Full save wipe / reset
- Save migration for older schemas

### Current Design Summary

The current game is broader than a basic clicker. It already functions as a layered incremental management game with:

- active clicking
- passive economy growth
- human and machine progression ladders
- research gating
- power/infrastructure constraints
- influence/lobbying systems
- repeatable scaling systems
- prestige meta progression
- offline and save support

## 3. Progression and Unlock Map

This section describes the current practical progression path and major gates.

### Early Game Flow

1. Start with manual trading only
2. Buy `Better Terminal`
3. Reach `Recruiter`
4. Unlock and hire `Interns`
5. Build toward `Junior Trader Program`
6. Unlock and hire `Junior Traders`

### Midgame Human Growth

7. Reach 5 Junior Traders
8. Unlock `Senior Recruitment`
9. Hire `Senior Traders`
10. Reach 5 Senior Traders
11. Branch into:
   - `Algorithmic Trading`
   - `Prop Desk Operations`

### Human Institutional Expansion

12. Unlock and build `Prop Desks`
13. Reach 3 Prop Desks
14. Unlock `Institutional Desks`
15. Build at least 1 `Server Room`
16. Build `Institutional Desks`
17. Reach 2 Institutional Desks
18. Unlock `Hedge Fund Strategies`
19. Build at least 1 `Data Centre`
20. Build `Hedge Funds`
21. Reach 1 Hedge Fund
22. Unlock `Investment Firms`
23. Build at least 1 `Cloud Infrastructure`
24. Build `Investment Firms`

### Research Staffing Branch

This branch starts after `Recruiter`.

1. Hire `Intern Scientists`
2. Unlock `Junior Scientists`
3. Hire `Junior Scientists`
4. Unlock `Senior Scientists`
5. Hire `Senior Scientists`
6. Use RP income to drive major research unlocks and RP-funded repeatables

### Machine / Infrastructure Branch

1. Reach 5 Senior Traders
2. Unlock `Algorithmic Trading`
3. Deploy `Rule-Based Bots`
4. Unlock `Power Systems Engineering`
5. Expand beyond starter rack into `Server Rooms`
6. Reach 5 Rule-Based Bots
7. Unlock `Data Centre Systems`
8. Build `Data Centres`
9. Deploy `ML Trading Bots`
10. Reach 3 ML Trading Bots
11. Unlock `AI Trading Systems`
12. Build `Cloud Infrastructure`
13. Deploy `AI Trading Bots`

### Lobbying Branch

1. Unlock `Power Systems Engineering`
2. Unlock `Regulatory Affairs`
3. Hire `Senators`
4. Generate `Influence`
5. Purchase policies across:
   - Labor
   - Energy
   - Market
   - Technology
6. Buy influence-funded repeatables for cost reduction and strategic scaling

### Prestige Gate

Prestige currently requires:

- Lifetime cash >= 4,000,000
- At least 1 Rule-Based Bot

When prestige becomes available:

1. Preview reputation gain
2. Plan prestige purchases
3. Confirm reset
4. Keep permanent prestige progress
5. Restart with stronger long-run bonuses and optional seed cash

### Phase Summary

The currently implemented phase structure is:

1. Solo Trader
   - Manual trading
   - First manual upgrades
   - Reach Recruiter

2. Junior Desk
   - Interns
   - Junior Traders
   - Early desk upgrades

3. Firm Growth
   - Senior Traders
   - Human desk scaling
   - Research branch expansion

4. Bot Era
   - Rule-Based Bots
   - Power infrastructure growth
   - ML / AI machine ladder
   - Institutional human ladder expansion
   - Lobbying opens up

5. Prestige Decision
   - Accumulate enough lifetime cash
   - Reach at least one bot
   - Convert run into permanent Reputation progression

## Short Assessment

The current playable build already supports a substantial incremental game loop with several interconnected progression layers. The most obvious unfinished area is not the main economy loop, but expansion/polish content:

- the disabled Commodities branch
- more settings controls
- optional meta systems like achievements or events

The core game arc itself is present and functional.
